This Movie Recommender system features a high-performance FastAPI backend and an interactive Streamlit frontend. It utilizes Scikit-learn to compute content-based recommendations using TF-IDF vectorization and cosine similarity on local datasets. Seamless integration with the TMDB API provides real-time movie details, extensive metadata, and high-quality posters. The architecture is decoupled, scalable, and asynchronous, ensuring a fluid user experience. Designed for cinematic discovery, it offers personalized suggestions and detailed movie insights across a robust, modern technology stack.
